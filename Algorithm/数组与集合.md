## 最长连续序列 - 哈希集合

【描述】

```text
给定一个未排序的整数数组 nums，找出其元素能组成最长连续序列的长度（不重复），要求时间复杂度为 O(n)。
例如：nums = [100,4,200,1,3,2]，最长连续子序列为 [1,2,3,4]，长度为 4。
```

【解释】

由于不能重复，所以先用一个 Set 存储所有元素进行去重，然后遍历每个元素。

假设连续子序列是 x, x+1, x+2, x+3，遍历的顺序是未知的，如果先遍历到 x+2，需要内循环同时向前和向后搜索，而且为了避免重复遍历，还需要额外的数据结构记录每个元素是否遍历过。

因此，只有当元素是连续子序列的开头元素才进入内循环，即集合不存在 x - 1，这样子既解决了双向搜索，还利用连续性规避了重复遍历。

【代码】

```java
public int longestConsecutive(int[] nums) {
    if (nums == null || nums.length == 0) {
        return 0;
    }
    
    Set<Integer> set = new HashSet<>();
    for (int num : nums) {
        set.add(num);
    }
    
    int longest = 0;
    for (int num : set) {
        if (!set.contains(num - 1)) {
            int currentNum = num;
            int currentLen = 1;
            
            while (set.contains(currentNum + 1)) {
                currentNum++;
                currentLen++;
            }
            
            longest = Math.max(longest, currentLen);
        }
    }
    
    return longest;
}
```

## 前 K 个高频元素 - 桶数组

【描述】

```text
text给定一个非空的整数数组，返回其中出现频率前 k 高的元素。要求时间复杂度优于 O(n log n)。
例如：nums = [1,1,1,2,2,3]，k = 2，返回 [1,2]。
```

【解释】

使用哈希表统计每个元素的出现频率，由于频率最大不会超过数组长度 n，可以创建一个 n+1 的频率数组，下标表示频率，值为具有该频率的元素列表。

因此先遍历哈希表，将每个元素放入对应频率的桶中，再从高频到低频遍历桶，取出前 k 个元素即可。

【代码】

```java
public int[] topKFrequent(int[] nums, int k) {
    // 1. 统计每个元素出现的频率
    Map<Integer, Integer> freqMap = new HashMap<>();
    for (int num : nums) {
        freqMap.put(num, freqMap.getOrDefault(num, 0) + 1);
    }
    
    // 2. 创建桶数组，下标表示频率
    List<Integer>[] bucket = new List[nums.length + 1];
    for (Map.Entry<Integer, Integer> entry : freqMap.entrySet()) {
        int freq = entry.getValue();
        if (bucket[freq] == null) {
            bucket[freq] = new ArrayList<>();
        }
        bucket[freq].add(entry.getKey());
    }
    
    // 3. 从高频到低频收集结果
    int[] result = new int[k];
    int count = 0;
    for (int i = bucket.length - 1; i >= 0 && count < k; i--) {
        if (bucket[i] != null) {
            for (int num : bucket[i]) {
                result[count++] = num;
                if (count == k) {
                    return result;
                }
            }
        }
    }
    
    return result;
}
```

## 第 K 大的元素 - 快速排序

【描述】

```text
给定一个整数数组 nums 和整数 k，请返回数组中第 k 个最大的元素。注意：这里是指排序后的第 k 个最大元素，而不是第 k 个不同的元素。
例如：nums = [3,2,1,5,6,4]，k = 2，返回 5。
```

【解释】

快速排序是选取当前数组最右边的值作为基准值 pivot，并维护一个 store 指针，目标是让 store 之前的值都是小于 pivot 的值，store 之后的值都是大于 pivot 的值

1. 遍历 i 从 left 到 right - 1，当 i 位置的值小于 pivot，则交换 store 和 i，并递增 store。
2. 遍历完成后，交换 store 和 right，就可以保证 store 位置一定就是 pivot 在排序好的数组的最终位置

【代码】

```java
class Solution {
    public int findKthLargest(int[] nums, int k) {
        int n = nums.length;
        int target = n - k;
        int left = 0;
        int right = n - 1;

        while (left < right) {
            int pivot = partition(nums, left, right);
            if (pivot == target) return nums[pivot];
            if (pivot < target) left = pivot + 1;
            if (pivot > target) right = pivot - 1;
        }

        return -1;
    }

    private int partition(int[] nums, int left, int right) {
        int pivot = nums[right];
        int store = left;
        for (int i = left; i < right; i++) {
            if (nums[i] < pivot) {
                swap(nums, store, i);
                store++;
            }
        }
        swap(nums, store, right);
        return store;
    }

    private void swap(int[] nums, int i, int j) {
        if (i == j) return;
        int temp = nums[i];
        nums[i] = nums[j];
        nums[j] = temp;
    }
}
```

## 滑动窗口最大值 - 单调递减队列

【描述】

```text
给定一个整数数组 nums 和一个大小为 k 的滑动窗口，窗口从数组最左端移动到最右端，每次向右移动一位。返回每个窗口中的最大值。
例如：nums = [1,3,-1,-3,5,3,6,7]，k = 3，返回 [3,3,5,5,6,7]。
```

【解释】

使用双端队列来维护窗口内的元素下标，之所以维护下表是因为可以根据索引差来判断是否处于窗口内，值可以通过索引回到原数组查找。

如果这个最大值是队头，那么只需 O(1) 就可以获取，如果这个最大值离开窗口了，仍需要保证新的队头是最大值，因此窗口内应该是单调递减的。

因此可以遍历数组下标 i，让 i 是窗口最后一个元素的下标，则窗口第一个元素的下标则是 i - k + 1：

1. 判断队头元素的下标是否还在窗口内，即是否 >= i - k +1，如果不在则移出；
2. 当前元素值为 num，从队尾往前移除所有 <= num 的值，确保窗口内的元素值是单调递减的；
3. 将当前元素加入队尾；
4. 如果 i >= k-1，则表示开始形成窗口，此时队首元素可以保证是当前窗口的最大值。

```java
class Solution {
    public int[] maxSlidingWindow(int[] nums, int k) {
        int n = nums.length;
        int[] ans = new int[n - k + 1];
        Deque<Integer> dq = new ArrayDeque<>();

        for (int i = 0; i < n; i++) {
            int num = nums[i];

            if (!dq.isEmpty() && dq.peekFirst() <= i - k) {
                dq.pollFirst();
            }

            while (!dq.isEmpty() && nums[dq.peekLast()] <= num) {
                dq.pollLast();
            }

            dq.addLast(i);

            if (i >= k - 1) {
                ans[i - k + 1] = nums[dq.peekFirst()];
            }
        }

        return ans;
    }
}
```

## 无重复字符的最长子串 - 滑动窗口

【描述】

```text
给定一个字符串 s，请你找出其中不含有重复字符的最长子串的长度。
例如：s = "abcabcbb"，输出 3（最长子串是 "abc" 或 "bca" 或 "cab"）；s = "bbbbb"，输出 1；s = "pwwkew"，输出 3（"wke" 或 "kew"）。
```

【解释】

维护 `left` 指针和 `right` 指针作为滑动窗口，以及一个哈希表 `Map<Character, Integer>` 记录窗口内出现的字符和索引：

1. `right` 指针不断向右移动，遍历每个字符 `ch`
2. 如果 `ch` 在 `map` 中了，说明它可能会和当前窗口中的字符重复，拿到这个存在字符的索引 `index`，那么新的 `left` 应该至少移动到 `index + 1`
3. 然而我们并不会清除已经失效的字符，所以 index 可能已经在当前窗口的左侧，因此为了确保 index 单调递增，实际应该是 `left = Math.max(left, index + 1)`
4. 最后，更新当前字符的最新位置 `map.put(ch, right)`，并用 `right - left + 1` 得到当前窗口的长度来更新最大长度

【代码】

```java
class Solution {
    public int lengthOfLongestSubstring(String s) {
        Map<Character, Integer> map = new HashMap<>();

        int left = 0;
        int res = 0;

        char[] chars = s.toCharArray();

        for (int right = 0; right < chars.length; right++) {
            char ch = chars[right];

            if (map.containsKey(ch)) {
                int index = map.get(ch);
                left = Math.max(left, index + 1);
            }

            map.put(ch, right);
            res = Math.max(res, right - left + 1);
        }

        return res;
    }
}
```

## 字符串的字母异位词 - 滑动窗口

【描述】

```text
text给定两个字符串 s 和 p，找到 s 中所有 p 的字母异位词（字母相同但排列不同的子串）的子串起始索引。
例如：s = "cbaebabacd", p = "abc"，返回 [0, 6]（子串 "cba" 和 "bac" 都是 "abc" 的字母异位词）。
```

【解释】

异位词说白了就是不管顺序，只需要字母及其个数相同，那么判断两个字符串是否是字母异位词，就只需要判断它们的字符统计是否相等即可。因此，可以利用两个长度为 26 的数组记录每个字母出现的次数，然后利用 Arrays.equals(arr1, arr2) 就可以快速判断。
同时，由于是找 p 的字母异位词，所以长度是固定的，可以在 s 中维护一个相同长度的滑动窗口，每次滑动 1 格，将最左侧的字符在数组的次数 - 1，将最右侧的字符在数组的次数 + 1。
最后，只需要先初始化这个滑动窗口就好了，即先把 s 中 0~pLen 范围的字符加入数组中。

【代码】

```java
public List<Integer> findAnagrams(String s, String p) {
    List<Integer> result = new ArrayList<>();
    if (s == null || p == null || s.length() < p.length()) {
        return result;
    }
    
    int[] pCount = new int[26];
    int[] sCount = new int[26];
	  int pLen = p.length();
  	int sLen = s.length();
    
    for (int i = 0; i < pLen; i++) {
        sCount[s.charAt(i) - 'a']++;
        pCount[p.charAt(i) - 'a']++;
    }
    
    if (Arrays.equals(pCount, sCount)) result.add(0);
    for (int i = 0; i < sLen - pLen; i++) {
        sCount[s.charAt(i) - 'a']++;
        sCount[s.charAt(i + pLen) - 'a']--;
        if (Arrays.equals(pCount, sCount)) result.add(i+1);
    }
    
    return result;
}
```

## 最小覆盖子串 - 滑动窗口

【描述】

```text
给你一个字符串 s 和一个字符串 t，请在 s 中找到包含 t 全部字符（包括重复字符）的最短连续子串，并返回该子串。如果不存在则返回空字符串。假设 s 和 t 只包含英文字母。
例如：s = "ADOBECODEBANC", t = "ABC"，输出 "BANC"
```

【解释】

首先通过 `right` 指针找到第一个满足条件的窗口。这个窗口不一定是最短的，例如窗口 `"BEBADC"` 已经包含了目标字符 `"BAC"`，但它可以继续缩短为 `"BADC"`，仍然包含 `"BAC"`。因此，当窗口已经满足条件后，`right` 边界暂时固定，接下来要不断移动 `left` 边界，尝试去掉窗口左侧多余字符，使窗口尽可能短。

当 `left` 收缩到某个位置后，如果再去掉当前字符会导致窗口不再覆盖 `t`，例如从 `"BADC"` 缩到 `"ADC"` 后缺少 `'B'`，此时说明当前窗口已经缩到极限，不能继续收缩，需要重新移动 `right`，向右扩展窗口，直到再次满足条件。

因此，这道题的核心是**：`right` 指针负责扩张窗口，直到窗口覆盖 `t`，`left` 指针负责收缩窗口，直到窗口刚好不再覆盖 `t`，在每一次窗口覆盖 `t` 的时候，尝试更新最短答案。**

使用一个 `needCount` 数组记录当前窗口相对于 `t` 还缺哪些字符、各缺几个；使用 `needSize` 记录当前窗口总共还缺多少个字符

1. 初始化时，遍历 `t`，令 `needCount[c]++`，表示目标字符串需要字符 `c`，同时令 `needSize = t.length()`，表示一开始窗口为空，总共缺少 `t.length()` 个字符
2. 移动 `right` 扩张窗口，如果进入前 `needCount[c] > 0`，说明当前窗口确实还缺这个字符，因此 `needSize--`
3. 当 `needSize == 0` 时，说明当前窗口已经包含了 `t` 所需的全部字符及数量，此时开始移动 `left` 收缩窗口
4. 移动 `left` 收缩窗口，先执行 `needCount[c]++`，表示窗口中少了一个字符 `c`，如果加完后 `needCount[c] > 0`，说明移出这个字符后，窗口重新缺少一个必要字符，因此 `needSize++`
5. 当 `needSize != 0` 时，说明当前窗口不再包含 `t` 所需的全部字符及数量，此时收缩停止，继续移动 `right`

有一个很重要的注意事项：对于不在 `t` 中的字符，它的 `needCount` 初始值为 `0`，当它被 `right` 纳入窗口时，`needCount[c]--`，变成负数，当它被 `left` 移出窗口时，`needCount[c]++`，最多回到 `0`。因此，不在 `t` 中的字符对应的 `needCount[c]` 永远不会大于 `0`，不会触发 `needSize--` 和 `needSize++`，因此不会干扰窗口是否满足条件的判断

【代码】

```java
class Solution {
    public String minWindow(String s, String t) {
        int[] needCount = new int[128];
        for (char c : t.toCharArray()) needCount[c]++;

        int n = s.length();
        int needSize = t.length();

        int left = 0;
        int right = 0;

        int start = 0;
        int minLen = n + 1;
        
        while (right < n) {
            char rightChar = s.charAt(right);
            if (needCount[rightChar] > 0) needSize--;
            needCount[rightChar]--;
            right++;

            while (left < n && needSize == 0) {
                int curLen = right - left;
                if (curLen < minLen) {
                    start = left;
                    minLen = curLen;
                }

                char leftChar = s.charAt(left);
                needCount[leftChar]++;
                if (needCount[leftChar] > 0) needSize++;
                left++;
            }
        }

        return minLen == n + 1 ? "" : s.substring(start, start + minLen);
    }
}
```

## 顺时针旋转数组 - 跳跃遍历

【描述】

```text
给定一个 n × n 的二维矩阵 matrix 表示一个图像，请你将图像顺时针旋转 90 度。要求原地修改矩阵，不能使用额外的矩阵存储。
例如：输入：matrix = [[1,2,3],[4,5,6],[7,8,9]]，输出：[[7,4,1],[8,5,2],[9,6,3]]
```

【解释】

核心思想是将矩阵看作一层一层的“圈”，从外圈向内圈逐层处理，每一层中，将上、右、下、左四条边上的对应元素进行四次循环交换，同时每一条边都将顶点作为起点，左上顶点往右遍历，右上顶点往下遍历，右下顶点往左遍历，左下顶点往上遍历。

那么就只需要确定当前层的上、下、左、右的位置，就可以组合成左上、左下、右上、右下四个顶点位置了，然后再从 0 ～ width 遍历即可。

【代码】

```java
class Solution {
    public void rotate(int[][] matrix) {
        int n = matrix.length;
        int left = 0;
      	int top = 0;
        int right = n - 1;
      	int bottom = n - 1;
      	int width = n - 1;

        while (width >= 1) {
            for (int i = 0; i < width; i++) {
                int temp = matrix[top][left + i];
                matrix[top][left + i] = matrix[bottom - i][left];
                matrix[bottom - i][left] = matrix[bottom][right - i];
                matrix[bottom][right - i] = matrix[top + i][right];
                matrix[top + i][right] = temp;
            }

            left++;
          	top++;
            right--;
          	bottom--;
          	width -= 2;
        }
    }
}
```

## 柱状图中最大矩形 - 单调递增栈

【描述】

```text
给定 n 个非负整数，用来表示柱状图中各个柱子的高度。每个柱子彼此相邻，且宽度为 1。求在该柱状图中，能够勾勒出来的矩形的最大面积。
例如：heights = [2,1,5,6,2,3]，输出 10（高度为 5 和 6 的柱子构成 5×2=10 的矩形）。
```

【解释】

利用一个单调递增栈来存储柱子下标，遍历每个柱子 i，当柱子 i 的高度小于栈顶柱子高度时，**表示栈顶的柱子无法继续扩展它的宽度，以它为高度的柱子能到达的最右边的位置就是 i**。所以我们需要不断处理栈顶，清理栈现有的元素，直到剩余元素和新进来的元素依旧保持单调递增。

如果栈里面是 1 3 5 6，当前是 2：

- 如果以 6 为高，那么宽就是 5 的下标到 2 的下标
- 如果以 5 为高，那么宽就是 3 的下标到 2 的下标，而单调性确保了从 3 下标开始，后面的柱子高度都是大于 5 的
- 如果以 3 为高，那么宽就是 1 的下标到 2 的下标，而单调性确保了从 1 下标开始，后面的柱子高度都是大于 3 的

最后，栈可能还剩一些元素，可以假设最后一个柱子的高度为 0，从而确保可以处理所有元素。

【代码】

```java
class Solution {
    public int largestRectangleArea(int[] heights) {
        int n = heights.length;
        Deque<Integer> stack = new ArrayDeque<>();
        int ans = 0;

        for (int i = 0; i <= n; i++) {
            int curHeight = (i == n ? 0 : heights[i]);
            while (!stack.isEmpty() && curHeight < heights[stack.peek()]) {
                int height = heights[stack.pop()];
                int left = stack.isEmpty() ? -1 : stack.peek();
                int width = i - left - 1;
                ans = Math.max(ans, height * width);
            }
            stack.push(i);
        }

        return ans;
    }
}
```

## 接雨水 - 单调递减栈

【描述】

```text
给定 n 个非负整数表示每个宽度为 1 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。
例如：height = [0,1,0,2,1,0,1,3,2,1,2,1]，输出 6。
```

【解释】

使用柱子的高度单调递减，但存储的是柱子下标。遍历每个柱子，若当前柱高大于栈顶柱高时，由于单调栈的性质，栈顶下还有元素则代表栈顶柱子的左边肯定也有一个比栈顶柱子高的柱子，也就是说栈顶柱子的左右两边都有一个比自己高的柱子，就可以形成凹槽来接雨水。

令栈顶柱子为底柱，当前柱子作为右柱，栈顶的下一个柱子作为左柱，那么就有

- 凹槽宽度 = 右侧索引 - 左侧索引 - 1
- 凹槽高度 = min(左柱高度, 右柱高度) - 底柱高度
- 容积 = 凹槽宽度 * 凹槽高度

重复此过程直到栈为空或栈顶高度不小于当前高度，然后将当前索引入栈。

【代码】

```java
class Solution {
    public int trap(int[] height) {
        int n = height.length;
        if (n == 0) return 0;

        int ans = 0;
        Deque<Integer> dq = new ArrayDeque<>();
        for (int right = 0; right < n; right++) {
            while (!dq.isEmpty() && height[right] > height[dq.peek()]) {
                int bottom = dq.pop();
                if (dq.isEmpty()) break;
              
                int left = dq.peek();
                int width = right - left - 1;
                int tall = Math.min(height[left], height[right]) - height[bottom];
              
                ans += width * tall;
            }
            dq.push(right);
        }   

        return ans;
    }
}
```

## 和为 K 的子数组 - 前缀和

【描述】

```text
给定一个整数数组 nums 和一个整数 k，请你统计并返回该数组中和为 k 的连续子数组的个数。
例如：nums = [1, 1, 1]，k = 2，输出 2（子数组 [1,1] 出现在下标 [0,1] 和 [1,2]）。
```

【解释】

对任意一个子数组[i, j]，可以得到和 k = sum(i, j) = sum(0, j) + sum(0, i - 1)，而 sum(0, j) 和 sum(0, i - 1) 都是数组的前缀和，结构相同。

 i 从 0 到 n-1 遍历数组，用一个哈希表记录前缀和出现的次数 map(i, count)，若遍历到 i 的时候前缀和为 sum，那么此时可以得到 k 的次数就是差值前缀和的次数 map.get(sum - k) ，而当下前缀和的次数为 map.get(sum) + 1。

【代码】

```java
class Solution {
    public int subarraySum(int[] nums, int k) {
        Map<Integer, Integer> map = new HashMap<>();
        map.put(0, 1);

        int count = 0;
        int pre = 0;
        for (int num : nums) {
            pre += num;
            if (map.containsKey(pre - k)) {
                count += map.get(pre - k);
            }
            map.put(pre, map.getOrDefault(pre, 0) + 1);
        }

        return count;
    }
}
```

## 三数之和 - 双指针

【题目】

```text
给定一个包含 n 个整数的数组 nums，判断 nums 中是否存在三个元素 a, b, c，使得 a + b + c = 0？找出所有满足条件且不重复的三元组。注意：答案中不可以包含重复的三元组。
例如：nums = [-1, 0, 1, 2, -1, -4]，返回 [[-1, -1, 2], [-1, 0, 1]]。
```

【解释】

如果我们将数组升序排序，遍历选取的第一个元素 i，范围是 [0 ～ n - 3]，那么就可以执行一些剪枝操作

- 如果 nums[i] 已经大于 0，那么它后面的元素肯定也大于 0，且三数之和肯定也大于 0，即不需要内循环，也不需要再外循环，直接 break
- 如果 nums[i] + nums[i+1] + nums[i+2] 已经大于 0，那么此时最小的三个数加起来都大于 0，所以也不需要内循环，也不需要再外循环，直接 break
- 如果 nums[i] + nums[n - 1] + nums[n - 2] 已经小于 0，那么此时这个数加最大的两个数都 小于 0，所以不需要内循环，继续外循环即可，直接 continue
- 如果 nums[i] = nums[i - 1]，那么选定的第一个元素是相等的，内循环得到的答案是重复的，所以不需要内循环，继续外循环即可，直接 continue

以上都是针对外循环的剪枝，而内循环，可以继续利用升序的性质，维护两个指针 left 和 right，令 sum = nums[i] + nums[left] + nums[right]，则有

- 如果 sum > 0，说明和大了，应该 right--，则可以让 sum 变小
- 如果 sum < 0，说明和小了，应该 left++，则可以让 sum 变大
- 如果 sum = 0，我们需要找到最后的 left 和最前的 right，从而避免重复，然后将 nums[i], nums[left], nums[right] 三元组作为答案之一

【代码】

```java
class Solution {
    public List<List<Integer>> threeSum(int[] nums) {
        Arrays.sort(nums);
        int n = nums.length;
        List<List<Integer>> res = new ArrayList<>();

        for (int i = 0; i <= n - 3; i++) {
            if (nums[i] > 0) break;
            if (nums[i] + nums[i + 1] + nums[i + 2] > 0) break;
            if (i >= 1 && nums[i] == nums[i - 1]) continue;
            if (nums[i] + nums[n - 2] + nums[n - 1] < 0) continue;

            int left = i + 1;
            int right = n - 1;

            while (left < right) {
                int sum = nums[i] + nums[left] + nums[right];
                if (sum < 0) left++;
                if (sum > 0) right--;
                if (sum == 0) {
                    while (left < right && nums[left] == nums[left + 1]) left++;
                    while (left < right && nums[right] == nums[right - 1]) right--;
                    res.add(Arrays.asList(nums[i], nums[left], nums[right]));
                    left++;
                    right--;
                }
            }
        }

        return res;
    }
}
```

## 最短无序连续子数组 - 双指针

【描述】

```text
给你一个整数数组 nums，你需要找出一个连续子数组，如果对这个子数组进行升序排序，那么整个数组都会变为升序排序。请你找出符合题意的最短子数组，并输出它的长度。
例如：nums = [2,6,4,8,10,9,15]，输出 5（子数组 [6,4,8,10,9] 排序后整个数组变为升序）。
```

【解释】

如果存在一个子数组 `[left, right]`，那么可以分成三部分左中右，保证的是：

- 左 + 中 <= 右的左边界：Max[0, right] <= nums[right + 1]
- 中 + 右 >= 左的右边界：Min[left, n-1] >= nums[left - 1]

那么，利用这个性质，就可以确定 left 和 right：

- 从左往右扫描，维护区间 [0, i) 中的最大值 max，如果 nums[i] < max，说明 i 肯定不是右，只能是中的右边界
- 从右往左扫描，维护区间 (i, n - 1] 中的最小值 min，如果 nums[i] > min，说明 i 肯定不是左，只能是中的左边界

【代码】

```java
class Solution {
    public int findUnsortedSubarray(int[] nums) {
        int n = nums.length;

        int left = -1;
        int right = -1;

        int max = Integer.MIN_VALUE;
        for (int i = 0; i < n; i++) {
            if (nums[i] < max) {
                right = i;
            } else {
                max = nums[i];
            }
        }

        int min = Integer.MAX_VALUE;
        for (int i = n - 1; i >= 0; i--) {
            if (nums[i] > min) {
                left = i;
            } else {
                min = nums[i];
            }
        }

        return right == -1 ? 0 : right - left + 1;
    }
}
```

## 跳跃游戏 - 贪心

【描述】

```text
给定一个非负整数数组 nums，你最初位于数组的第一个下标。数组中的每个元素代表你在该位置可以跳跃的最大长度。判断你是否能够到达最后一个下标。
例如：nums = [2,3,1,1,4]，返回 true；nums = [3,2,1,0,4]，返回 false。
```

【解释】

从最右边开始，让 target 表示从终点反跳能跳回的最左边位置，那么如果站在 i 能跳到 target，那么就可以更新 target = i，最后看能到达最左边的位置是否为 0

【代码】

```java
class Solution {
    public boolean canJump(int[] nums) {
        int target = nums.length - 1;
        for (int i = nums.length - 2; i >= 0; i--) 
            if (i + nums[i] >= target) target = i;
        return target == 0;
    }
}
```

## 合并区间 - 贪心

【描述】

```text
以数组 intervals 表示若干个区间的集合，其中每个区间为 [starti, endi]。请你合并所有重叠的区间，并返回一个不重叠的区间数组，该数组需恰好覆盖输入中的所有区间。
例如：intervals = [[1,3],[2,6],[8,10],[15,18]]，输出 [[1,6],[8,10],[15,18]]。
```

【解释】

任意两个区间 [leftA, rightA], [leftB, rightB] 只有以下三种情况

- leftA < rightA < leftB < rightB：两个区间完全不重叠
- leftA < leftB < rightB < rightA：一个区间包含另一个区间
- leftA <= leftB <= rightA <= rightB：两个区间重叠

我们可以将数组按找 left 排序，那么就可以从左往右按顺序进行遍历 i，对于当前的 left 和 right，基于上述三种情况：

- 情况一：left 和 right 成为答案的一个区间
- 情况二：left 和 right 不变，新区间被包含
- 情况三：只需要更新 right 为最新的右区间就能实现区间合并

【代码】

```java
class Solution {
    public int[][] merge(int[][] intervals) {
        Arrays.sort(intervals, (a, b) -> Integer.compare(a[0], b[0]));
        List<int[]> list = new ArrayList<>();

        int left = intervals[0][0];
        int right = intervals[0][1];

        for (int i = 1; i < intervals.length; i++) {
            if (intervals[i][1] <= right) {
              	continue;
            } else if (intervals[i][0] > right) {
                list.add(new int[]{left, right});
                left = intervals[i][0];
                right = intervals[i][1];
                continue;
            } else {
                right = intervals[i][1];
                continue;
            }
        }

        list.add(new int[]{left, right});
        return list.toArray(new int[list.size()][]);
    }
}
```

## 任务调度器 - 贪心

【描述】

```text
给定一个字符数组 tasks，表示 CPU 需要执行的任务列表，每个字母代表一种任务。每个任务需要 1 个单位时间执行，但两个相同种类的任务之间必须至少间隔 n 个单位时间（冷却时间）。可以合理安排任务的执行顺序，也可以在执行过程中插入空闲（idle）。求完成所有任务所需的最短时间。
例如：tasks = ["A","A","A","B","B","B"]，n = 2，输出 8（执行序列：A → B → idle → A → B → idle → A → B）。
```

【解释】

我们应该总是先执行剩余次数最多的任务去执行，这样才能保证它在冷却过程中可以去安排别的任务，否则拖到最后它的冷却过程只能 idle，无法有效利用所有时间。

因为 tasks 确保是单个字母，所以可以用 int count[26] 来统计每个任务的次数，如果当前任务在时间点 i 执行了，那么它下一次可以执行的时间点为 i + n + 1，可以用一个 int nextValidTime[26] 来记录每个任务允许执行的最早时间点。

所以遍历时间 time，当前应该执行的任务 task 需要确保以下两个条件：

- nextValidTime[task] <= time
- task 是 count 中最小的

如果能确定 task，就可以让任务总数量 - 1，当然存在无法确定 task 的情况，那么只能 idle，任务总数量不变。

【代码】

```java
class Solution {
    public int leastInterval(char[] tasks, int n) {
        int[] count = new int[26];
        int[] nextValidTime = new int[26];
        for (char task : tasks) {
            count[task - 'A']++;
        }

        int remain = tasks.length;
        int time = 0;

        while (remain > 0) {
            int best = -1;
            for (int i = 0; i < 26; i++) {
                if (count[i] == 0) continue;
                if (time < nextValidTime[i]) continue;
                if (best == -1 || count[i] > count[best]) best = i;
            }

            if (best != -1) {
                count[best]--;
                remain--;
                nextValidTime[best] = time + n + 1;
            }

            time++;
        }
        
        return time;
    }
}
```

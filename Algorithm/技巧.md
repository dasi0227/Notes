# 技巧

排序集合：`Collections.sort(list);`

移除对象：`list.remove(Integer.valueOf(2));`

移除所有符合条件元素：`list.removeIf(x -> x < 0);`

初始化列表：`List<Integer> list = new ArrayList<>(Arrays.asList(1, 2, 3, 4));`

初始化数组：`char[][] matrix = {{'1','0','1'}, {'1','0','1'}};`

截取列表：`list.subList(from, to)`

截取数组：`Arrays.copyOfRange(arr, from, to)`

倒序排列集合：`Collections.sort(list, Collections.reverseOrder());`

打印一维数组：`System.out.println(Arrays.toString(arr));`

打印二维数组：`System.out.println(Arrays.deepToString(matrix));`

修改字符串字符：`sb.setCharAt(index, char);`

删除字符串字符：`sb.deleteCharAt(index);`

删除字符串区间：`sb.delete(from, to);`

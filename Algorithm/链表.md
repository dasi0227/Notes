## LRU

【描述】

```text
请你设计并实现一个满足 LRU (最近最少使用) 缓存的数据结构 LRUCache。
以正整数 capacity 作为容量初始化，提供 get(key) 和 put(key, value) 方法，必须以 O(1) 的平均时间复杂度运行。
如果 get 的 key 不在缓存中则返回 -1。
```

【解析】

LRU 需要利用一个 Map + 双向链表实现，Map 负责检查是否命中缓存，而双向链表负责维护最近关系

- 添加的节点总是放到链表最后
- 当命中缓存的时候，需要将节点移动到链表最后，必须用双向链表，否则无法实现 O(1)
- 当缓存满的时候，需要删除链第一个的节点

【代码】

```java
class LRUCache {
    class Node {
        int key, value;
        Node prev, next;
        Node(int key, int value) {
            this.key = key;
            this.value = value;
        }
    }

    private Map<Integer, Node> map = new HashMap<>();
    private Node dummy = new Node(-1, -1);
    private Node tail = dummy;
    private int capacity;

    public LRUCache(int capacity) {
        this.capacity = capacity;
    }
    
    public int get(int key) {
        Node node = map.get(key);
        if (node == null) return -1;
        moveToTail(node);
        return node.value;
    }
    
    public void put(int key, int value) {
        Node node = map.get(key);
        if (node != null) {
            node.value = value;
            moveToTail(node);
        } else {
            node = new Node(key, value);
            map.put(key, node);
            addToTail(node);
            if (map.size() > capacity) {
                removeHead();
            }
        }
    }

    private void moveToTail(Node node) {
        if (node == tail) return;
        node.prev.next = node.next;
        if (node.next != null) node.next.prev = node.prev;
        else tail = node.prev;
        addToTail(node);
    }

    private void addToTail(Node node) {
        tail.next = node;
        node.prev = tail;
        node.next = null;
        tail = node;
    }

    private void removeHead() {
        Node head = dummy.next;
        map.remove(head.key);
        dummy.next = head.next;
        if (head.next != null) head.next.prev = dummy;
        else tail = dummy;
    }
}
```



## 入环节点

【描述】

```text
给定一个链表的头节点 head，返回链表开始入环的第一个节点，如果链表无环，则返回 null。
```

【解析】

让快指针和慢指针从头开始走，慢指针每次走 1 步，快指针每次走 2 步。
当它们相遇时，设从头到入环节点的距离为 x，慢指针在环内走了 y，环长为 z。
相遇时，快指针肯定比慢指针多走了 n 圈，而快指针的步数又是慢指针的两倍。
因此，可得方程 2(x + y) = x + y + nz，即 x = nz - y。
也就是说，从头节点走 nz - y 步正好到达入环节点，同时慢指针在环内走了 y 步，再走 nz - y 步也正好到达入环节点。
因此让头指针和慢指针每次只走 1 步，相遇的时候，就是入环节点。

【代码】

```java
public class Solution {
    public ListNode detectCycle(ListNode head) {
        ListNode slow = head, fast = head;
        while (fast != null && fast.next != null) {
            slow = slow.next;
            fast = fast.next.next;
            if (slow == fast) {
                ListNode ptr = head;
                while (ptr != slow) {
                    ptr = ptr.next;
                    slow = slow.next;
                }
                return ptr;
            }
        }
        return null;
    }
}
```
